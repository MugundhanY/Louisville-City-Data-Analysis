# Louisville City Data Analysis

This repository contains Python code for conducting a comprehensive data analysis on the city of Louisville, comparing the years 2008 and 2016. The analysis involves using Sweetviz and Pandas Profiling for insightful visualizations and statistical summaries.

## Requirements

- Python 3.x
- Pandas
- Sweetviz
- Pandas Profiling

Install the required libraries using:

```bash
pip install pandas sweetviz pandas-profiling
